4. 2017 Denver Inversion¶

4.1. Namelist,Datasets and Scripts¶

4.1.1. Namelist¶

The case runs are initialized at 00z Oct 16, 2017 with 168 hours forecasting. The corresponding namelist options that need to be changed are listed below.

Table Namelist options¶

Options

Value

RUN_STARTDATE

2017-10-16

START_TOD

0

STOP_OPTION

168

STOP_N.

nhours

4.1.2. Datasets¶

Initial condition (IC) files are created from GFS reanalysis dataset in nemsio format. These files should be put in the /run/INPUT directory.

  • Download the 2019 Denver Inversion case initial condition files

Sounding profiles can be downloaded from the University of Wyoming.

4.1.3. Scripts¶

An example Python script is provided below to do the Skew-T Log-P plots for the observed and simulated sounding profiles. Indices for sounding analysis (e.g. lapse rate) are also calculated and shown at the bottom of the plot. This script is adapted from SHARPpy. Data format needs to comply with SHARPpy requirements.

  • Download the example script for Skew-T Log-P plot
